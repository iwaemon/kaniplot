## Issues Found

1. **No README or documentation** (High) — New users have no way to discover how to use kaniplot without reading source code
2. **No --help flag** (High) — CLI provides no usage information
3. **Replot concatenates SVGs** (Medium) — When outputting to file, replot appends a second SVG instead of replacing
4. **Discontinuous function rendering** (Medium) — Lines are drawn through singularities (1/x at x=0, tan(x) at pi/2) instead of breaking
5. **Spec says PNG default, implementation uses SVG** (Low) — Minor spec/implementation mismatch
6. **Tick label trailing zeros** (Low) — Tick labels show unnecessary trailing zeros (e.g., "2.00" instead of "2")
7. **Legend line sample doesn't match data line width** (Low) — When linewidth is set, the legend sample line uses default width

## Suggestions

1. **Create a README.md** with installation instructions, basic usage examples, and supported commands
2. **Add --help / -h flag** to the CLI with usage synopsis
3. **Fix replot file output** — truncate/overwrite the output file on replot instead of appending
4. **Add discontinuity detection** — break polylines when consecutive y-values span a large range relative to the plot bounds
5. **Clean up tick formatting** — strip trailing zeros from tick labels
6. **Sync legend line width** with data series line width
