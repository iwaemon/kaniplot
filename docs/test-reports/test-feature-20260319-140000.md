## Per-Feature Details

### Basic data file plotting
- **Role:** Researcher plotting experimental data
- **What they tried:** `plot "data.dat" with lines`
- **Discoverability:** Not documented in README
- **Setup:** Build works
- **Functionality:** Polyline with 5 points rendered correctly
- **Friction points:** No docs for data file syntax
- **Doc suggestions:** Add data file section to README

### Using column selection
- **Role:** Researcher selecting specific columns
- **What they tried:** `plot "data.dat" using 1:3 with lines`
- **Functionality:** Y-axis correctly shows column 3 values (100-300)

### Using column expression
- **Role:** Researcher applying transformations
- **What they tried:** `plot "data.dat" using 1:($2*100) with lines`
- **Functionality:** Column expression evaluated correctly (0.5*100=50, etc.)

### Index block selection
- **Role:** Researcher with multi-dataset files
- **What they tried:** `plot "data.dat" index 1 with lines`
- **Functionality:** Second block (100-300) selected correctly

### Every row stride
- **Role:** Researcher downsampling large datasets
- **What they tried:** `plot "data.dat" every 2 with lines`
- **Functionality:** 3 points from 6 rows (rows 0, 2, 4)

### Comments and missing values
- **Role:** Researcher with messy data
- **What they tried:** File with `#` comments and `?` missing values
- **Functionality:** Comments skipped, missing row excluded, polyline connects valid points

### Mixed data + expression
- **Role:** Researcher comparing data to model
- **What they tried:** `plot "data.dat" with points, x**2 with lines`
- **Functionality:** Both series rendered with correct legend entries

### Error handling
- **Role:** User with typo in filename
- **What they tried:** Plot nonexistent file
- **Functionality:** Clean error message, no crash
